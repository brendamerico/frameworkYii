  2. O QUE É O FRAMEWORK YII?
    • "simples e evolutivo" / Yes It Is !
    • O Yii é um framework em PHP que usa componentes para o desenvolvimento de aplicações Web sejam elas pequenas, médias ou grandes.
    • Yii é um framework PHP de alta performance baseado em componentes para desenvolvimento rápido de aplicações web modernas.
    • A proposta do Yii é fazer aplicações Web complexas de maneira simples, economizando tempo e recursos do servidor. O Yii oferece uma grande experiência de desenvolvimento para o programador e uma boa usabilidade da aplicação para os usuários finais.
    • O Yii é open-source ou código aberto e é escrito totalmente na versão 5 da linguagem PHP.
    • O Yii apresenta a arquitetura MVC na construção dos seus projetos e sua estrutura de arquivos e pastas muito bem organizada, separando o core do framework com o core da aplicação.
    • O Yii apresenta suporte para todas as tecnologias do mercado atual.

  3. PARA QUE SERVE
    • O Yii é um framework de programação web genérico que pode ser utilizado para o desenvolvimento de todo tipo de aplicações web usando PHP.
    • Devido a sua arquitetura baseada em componentes e suporte sofisticado a caching, ele é adequado para o desenvolvimento de aplicações de larga escala, como sistemas de gerenciamento de conteúdo (CMS), projetos de e-commerce, Web services RESTful, entre outros.

  4. VANTAGENS E DESVANTAGENS

      5. VANTAGENS
    • Web de código aberto;
    • Orientado a objetos;
    • Utilização de design patterns;
    • Baseado em MVC;
    • Documentação atualizada, bem organizada e muito exemplificada;
    • Fácil aprendizado;
    • Features avançadas que outros frameworks não possuem nativamente.

      DESVANTAGENS
    • Documentação e algumas referências todas em inglês;
    • O processo de instalação confunde alguns usuários iniciantes, levando-os a buscar um tutorial sobre a instalação.

  5. QUAIS RECURSOS OFERECE?
    • Implementa o padrão de arquitetura MVC e promove a organização do código nesse padrão.
    • Fornece construtores de consultas e ActiveRecord, suporte ao desenvolvimento de APIs RESTful, suporte a caching de múltiplas camadas, etc.
    • É extensível, podendo personalizá-lo ou substituir quase todas as partes do código central.
    • Também pode aproveitar-se de sua arquitetura para usar ou desenvolver extensões que podem ser redistribuídas.
    • O framework Yii apresenta uma alta performance.

    • Yii é apoiado por equipes de desenvolvedores do código central e por uma ampla comunidade de profissionais constantemente contribuindo com o desenvolvimento do Yii.
